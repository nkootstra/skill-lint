# Agents

## Build

This is a GitHub Action built with TypeScript and bundled with `@vercel/ncc`.

- **Always run `npm run build` before committing** when you change any file under `src/`. The `dist/` directory must be checked in and kept in sync with the source. CI will fail if `dist/` is out of date.
- Run `npm test` to execute tests with Vitest.
- Run `npm run typecheck` to verify types.

## Project structure

- `src/` — TypeScript source
- `dist/` — Compiled bundle (committed, generated by `npm run build`)
- `action.yml` — GitHub Action metadata
